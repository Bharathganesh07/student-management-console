\# Student Management Console Application (Java + MySQL)



A console-based student management system built using Java and MySQL.



\## Features

\- Add student

\- View students

\- Search student

\- Update student

\- Delete student



\## Technologies Used

\- Java (JDBC)

\- MySQL

\- IntelliJ IDEA



\## How to Run

1\. Create database `studentdb`

2\. Create `students` table

3\. Update DB credentials in code

4\. Run Main.java
